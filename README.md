# Tienda Segunda Mano


## Propósito
Sitio web de ejemplo para la entrega del proyecto del curso. Muestra un catálogo de artículos de segunda mano, blog y formulario de contacto con validación.


## Tecnologías
- HTML5
- CSS3
- JavaScript (vanilla)


## Cómo ejecutar
1. Descargar o clonar este repositorio.
2. Abrir `index.html` en tu navegador o servirlo con un servidor local (recomendado: `live-server` o `python -m http.server`).


## Validación
Validar archivos HTML en https://validator.w3.org/ y CSS en https://jigsaw.w3.org/css-validator/.


## Notas
- Las imágenes de `assets/images/` deben ser añadidas antes de publicar.
- En un proyecto real, los datos de productos deberían venir de un backend o JSON externo.
